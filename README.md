Emailing in Node - Sep/10/2014
============================

Instructions to Run Examples
----------------------------
1. Clone the repo  
2. Run 'npm install' to download the dependencies  
3. a) Run 'node basicEmail.js'  
   b) Run 'node advancedEmail.js'  
4. Play around with different options
